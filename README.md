# Crystal Idle - Evolution

Webová idle hra optimalizovaná pro mobilní zařízení.

## Jak spustit
Otevřete `crystal-idle/index.html` ve svém prohlížeči. Není potřeba žádná instalace ani server.

## Funkce
- **Idle Produkce:** Generujte data pomocí serverů a satelitů.
- **Evoluce:** Dosáhněte limitu paměti pro zisk Hvězdy (soft reset).
- **Prestige Upgrady:** Vylepšete efektivitu pomocí získaných hvězd.
- **Offline Progress:** Hra ukládá stav automaticky.

## GitHub Pages
Tato složka je připravena pro nasazení na GitHub Pages.
Všechny cesty jsou relativní.

## Struktura kódu
- `game.js`: Matematika a herní logika.
- `ui.js`: Manipulace s DOMem (Smart Render).
- `storage.js`: Ukládání dat.
- `main.js`: Hlavní smyčka.
